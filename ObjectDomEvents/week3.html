Events

In the last section we saw how the DOM act as an interface that allows us to use JavaScript to interact with a web page. Another part of the DOM that provides the link between the web page and user interactions is called Events. Thus, this event occurs every time a user interacts with a web page, such as pressing a key, or moving a mouse. In this section we discuss Event listerner and The event object.

Events Listerners

Event listerner gives us the oppurtunity to be able to get a notification as soon as something happens with our program. Instead of us having to constantly check to see if an event has occurred, the event listener will let it know when the event happens, thus enabling us to then respond appropriately.

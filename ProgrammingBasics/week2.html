Functions, Variables and Parameters

A function in JavaScript is similar to a procedure, it is a set of statements that performs a task or calculates a value. Where there is some obvious relationship between the input and the output it should be able to take some input and return an output. This clearly makes a procedure to qualify as a function. We have many different types of functions. Here in this lesson we will see how functions is used to showcase understanding of loops, Conditional Statements, and Arrays. For a better understanding of functions we need to explain variables and parameters. All programs uses variables and parameters that contains values that the program will need.
Variable - In programming a variable is basically a box that contains something. It could be a number, text, or something else. We can define a variable using this format.
var varnameN = "valueN";
The keyword var enables the computer to know we are creating a new box. The = means that we are putting something in the box. To make it easier when we define variables with functions, strings, or even lists, you have to think about variables like this syntax above.
Parameters - I will like to define parameter in simple terms. In this context lets say a parameter is used to identify which variables will be used in the function. Again to make more sense we could say a parameter is a named variable passed into a function Think about parameter as variables used to import arguments into functions. we could call the variables doors into a function. The doors are labeled. Once information goes through the door it can be used in the function. It also lets the program pass information into the function.

All functions follow the same format and have the same basic parts.
function funcnameN(parameter1, parameter2,...){
statements;
return value;
}
  • function keyword - this lets the computer know you are starting a function
  • funcnameN - this unique identifier is used to call (or run) the function.
  • parameter list - this is used to identify what variables (information) will be fed into the function. when the function is called using its name, the value of the parameters can be assigned.
  • statements - this is where all the work of the function is done.
  • return value - this is the result of the function or what comes out of it. It is returned to the part of the program that called the function (or made it run)

Loops

Loops offer a quick and easy way to do something repeatedly, refer to a function where the statments are run multiple times while a certian condition is followed. For our example we are going to do a loop that will display a times table. The user will input the times table they want to see and how large they want it to be. When the user clicks the button, the computer will call the function to run. The statements inside the function thus calculate the times table, add each times table to what will be the output, until the users specificity is met in relation to the table size. As soon as the table size is met, the program will send the total times table to the computer for display. For clarity lets see what the code for this loop looks like below.
function timesTable(timesTableNum, tableSize){
var timesTableNum = parseInt(document.getElementById("timesTableNumInput").value);
var tableSize = parseInt(document.getElementById("timesTableSizeInput").value);
var ans = 0;
var i = 0;
var output = "";
while (i<=tableSize) {
ans = timesTableNum*i; output += timesTableNum + " x " + i + " = " + ans + <br>;
i++;}
document.getElementById("timesTableDiv").innerHTML = output; }

Logical Statements

in this section we are looking at logical conditions that allow you to control the flow of a program, it thus gives a logical condition telling the computer to evaluate if a condition is true. This is a natural occurrence that will observe in our everyday life. For example, in order to join a boys scout by yourself you have to be at least 12 years old and 80 inches tall. We can write a function to ask the user for their age and height to determine if they can join boys scout. If the applicant is over 80 inches tall but not yet 12years old they can still join with the consent of their parents. The function would look like this.
function canRide(age,height){
var age = parseInt(document.getElementById("riderAge").value);
var height = parseInt(document.getElementById("riderHeight").value);
var output = "";
if (age >= 12 && height >= 80){
output = "Hooray! You can join boys scout by yourself!";
}
else if (height >= 80){
output = "Not quite old enough, but don't give up! You can still join with the consent of your parents!";
}
else {
output = "I am sorry you must be over 80 inches to join, and 12 or older to join by yourself";
}
document.getElementById("joinDiv").innerHTML = output; }

Arrays

Arrays are a neat way of storing a list of data items under a single variable name and it is a special kind of variable that can hold more than one value. It allows us to store lists in one variable. Here are a few important things to understand about arrays.
Syntax - vararray_name = [item1, item2, item3,...]
Creating an Array - To create a new array we use "new" keyword like in the following example:
var join = new Array[ Boys Brigade, Church Lads Brigade, YMCA Boy Scouts]
Each item seperated by a comma is stored in a spot in the array. Each spot is numbered starting with 0. You can see any item in an array by using its number.
For example: join[0] = boys Brigade, join[1]= Church Lads Brigade
You can add new items to an existing array by "pushing" into the list.
For example: join.push( Chums Scout Patrols) would add " Chums Scout Patrols" to the last spot in our array.
Array Length - All arrays have a special property called "length" this tells us how many items are in the array.
For example: var join = [Boys Brigade, Church Lads Brigade, YMCA Boy Scouts, Chums Scout Patrols]; join.length = 4

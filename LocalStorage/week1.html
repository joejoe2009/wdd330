Week 1 - Local Storage API, Storing and Retrieving Simple Data, Arrays, Associative Arrays, and Objects

LocalStorage from my understanding persist across multiple sessions, thus help to store data so it is not lost when the application stops. LocalStorage enables one to show how to get, store and display usersâ€™ data. In simple terms utilizing localStorage is like going to a shopping cart website and add something to our cart, then come back to the site later to find the item still in our cart. Below is a syntax that will help us to better understand how best locastorage works.

Syntax

To save data
localStorage.setItem("itemName", "value")
Example:

localStorage.itemSet("username", "JoeLarge");
To access stored data
alert("itemName = " = localStorage.getItem("itemName"));
Example:
alert("username = " + localStorage.getItem("username"));
The alert output would be "username = JoeLarge"

Is it Supported

Having said all of this, it is important we know that web storage is not always supported, so before using web storage(local Storage) one must check to ensure that local Storage is available.
